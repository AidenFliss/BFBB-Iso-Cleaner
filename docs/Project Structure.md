## Project Structure

The project tree view is below
```
/
├── obj - Visual Studio Files
├── bin - Binary files
│   ├── Debug - Debug - builds for development / work in progress
│   └── Release - Release - builds for GitHub
├── Properties - Publish profiles for forking / making a one exe build file
├── .git - Stuff from GitHub Desktop
├── .gitattributes - Other GitHub stuff
├── LICENSE - License to the software / code
└── README.md - A read me file
```
